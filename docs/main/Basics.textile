h2. Nozioni base

Imposta, controlla, leggi, ripristina o elimina i cookie manualmente o tramite variabili HTTP.

p(alert-block warning). %(ui-icon ui-icon-alert).% According to the EU legislation you must warn your users if your website set cookies.

h2. Requisiti del plugin

Requisiti minimi di oui_cookie:

* "Textpattern CMS":http://textpattern.com/ 4.6+.

h2. Installazione

h3. Dall'interfaccia di amministrazione

# "Download":https://github.com/NicolasGraph/oui_cookie/releases il file del plug-in compilato o l'origine per compilare un file personalizzato.
# Incolla il contenuto del file del plug-in compilato sotto "Admin > Plugins":?event=plugin scheda e fare clic sul pulsante _Upload Aggiornare_.
# Conferma l'installazione del plug-in facendo clic sul pulsante _Install (installa)_ nella pagina di anteprima del plug-in.
# Abilita il plug-in.

h3. Mediante compositore

# dopo "installa compositore":https://getcomposer.org/doc/00-intro.md…

# indica il percorso selezionato:
@$ cd /path/to/your/textpattern/installation/dir@
# Se non è già stato fatto, blocca la tua versione di Txp:
@$ composer require textpattern/lock:4.6.2@, dove @4.6.2@ è la versione Txp in uso.
# Installa oui_cookie:
@$ composer require oui/oui_cookie@

h2. Aggiornare

h3. Dall'interfaccia di amministrazione

# Seguire le istruzioni di installazione sopra.

h3. Mediante compositore

bc. $ composer update oui/oui_cookie

h2. Disistalla

h3. Dall'interfaccia di amministrazione

# Seleziona la casella a sinistra della riga del plugin sotto il "Admin > Plugins":?event=plugin.
# apri l'elenco di selezione nella parte inferiore della tabella dei plugin e scegli _Delete (cancella)_.
# confirm the plugin deletion.

h3. Mediante compositore

bc. $ composer remove oui/oui_cookie
